linux命令使用 
      1.找出文件中重复的行,并显示重复次数 
        命令: cat xxx.txt | sort | uniq -c
      2.返回上次的目录: cd -
      3.切换到当前用户home目录: cd ~
      4.ctrl+a  //命令行下，光标称动到开头
      5.ctrl+e  //命令行下，光标移动结尾
      6.echo $$ //输出当前进程ID
      7.echo $! //输出最近执行的命令的进程ID
      8.[ -e file ] //测试文件是否存在
      9.[ -d dir ] //测试目录是否存在
      10. echo $? //输出上一次命令执行(最近执行的命令)的结果,0-正常,非0-不正常
      11. which -a cmd_name //会在PATH环境变量里面的目录寻找命令的绝对路径
      12. du -sh $dir_name //显示目录的磁盘占用空间
